The Titanic dataset contains 891 passengers and 12 features describing passenger information and survival status. Several columns contain missing values, especially Cabin, Age, and Embarked. The dataset includes both numerical features such as Age, Fare, and Passenger Class, as well as categorical features like Sex and Embarked. This initial exploration shows that data cleaning will be required before building any machine learning model. Overall, the dataset provides a solid foundation for learning data preprocessing and predictive modeling.
